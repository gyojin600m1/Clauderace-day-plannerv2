# 水泳資格級 判定アプリ（2026年度）

日本水泳連盟「2026年度 水泳資格表」（2026/04/10現在）をもとに、
**性別・年齢・種目・距離・タイム** を入力すると **何級か** を判定します。

## 使い方
- `swim.html` をダブルクリックしてブラウザで開くだけ（オフラインで動作）。
- スマホでも使えます。`swim.html` を端末に送って開いてください。

## 主な機能
- 入力すると即判定。**級・カテゴリ(AA/A/B)・次の上位級まであと何秒** を表示。
- その条件の **全15級の基準タイム一覧** を表示し、各級と自分のタイムの差（速い/遅い）も表示。該当級はハイライト。
- タイムは「28.50」「1:02.34」「62.34」のいずれの形式でも入力可。

## 判定ルール
- 級は速いほど上位（**15級が最上位、1級が最下位**）。
- カテゴリ: **AA = 15〜11級 / A = 10〜6級 / B = 5〜1級**（PDF原本の区分ラベルに基づく）。
- 入力タイムが各級の基準タイム **以下** なら、その級として判定。
- 1級の基準にも届かない場合は「**級外**」。
- 年齢区分: 19歳以上 / 17〜18 / 15〜16 / 14 / 13 / 12 / 11 / 10 / 9 / 8歳以下（年齢から自動判定）。

## ファイル構成
- `swim.html` … 完成アプリ（データ埋め込み済み・これ単体で動く）
- `data.json` … PDFから抽出した基準タイム（検証用・全フィールド）
- `data_arr.json` … 軽量化した基準タイム（配列形式・swim.htmlに埋め込み）
- `parse.py` … PDF → data.json 変換スクリプト
- `template2.html` … swim.html の元テンプレート（`/*DATA*/` にデータを差し込む）

## データ更新時（来年度版など）
1. 新しいPDFを `parse.py` の `PDF` パスに設定して `python3 parse.py`
2. `data_arr.json` を再生成（parse.py 内の COLS 順で配列化）
3. `template2.html` の `/*DATA*/` に `data_arr.json` を差し込んで `swim.html` を再生成

## 備考
- 元PDFの背泳ぎ100m（男子17〜18歳・8級）に小数1桁表記（`59.4`）の表記ゆれがありましたが、`59.40` として取り込んでいます。
- カテゴリ境界は、PDF原本の左側区分ラベル（AA=13級・A=8級・B=3級の位置＝各5級ずつ）に基づき **AA=15〜11 / A=10〜6 / B=5〜1** としています。
